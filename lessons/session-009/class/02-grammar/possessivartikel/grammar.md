# Gramática — Los posesivos (Possessivartikel): mein, dein, sein, ihr…

**Sesión 9 · roadmap A1 #12.** Se **califica** con `/grade`.
Ref. acumulada: `grammar/possessivartikel.md`.

---

## La idea grande (¡buenas noticias!)

Los posesivos (*mi, tu, su…*) se declinan **EXACTAMENTE como `ein-` y `kein-`** — las mismas terminaciones que ya dominas. Solo cambias la "raíz":

> **ein / kein / mein / dein / sein / ihr …** → **todas toman las mismas terminaciones.**

Así que si sabes *ein**en** Kaffee* y *kein**e** Suppe*, ya sabes *mein**en** Pullover* y *mein**e** Jacke*. 🎯

---

## Las raíces (según el "dueño")

| Dueño | Posesivo | Ejemplo |
|---|---|---|
| ich | **mein** | mein Hemd (mi camisa) |
| du | **dein** | dein Pullover |
| er / es | **sein** | sein Schuh (de él) |
| sie (ella) | **ihr** | ihr Kleid (de ella) |
| wir | **unser** | unser Korb |
| ihr (vosotros) | **euer** | euer Geschäft |
| sie (ellos/ellas) | **ihr** | ihre Jacke |
| Sie (usted, formal) | **Ihr** | Ihre Karte (con mayúscula) |

---

## Las terminaciones (= las de ein-/kein-)

| | Nominativo | **Akkusativo** |
|---|---|---|
| masculino | mein | **meinen** ← ¡cambia! |
| femenino | mein**e** | mein**e** *(igual)* |
| neutro | mein | mein *(igual)* |
| plural | mein**e** | mein**e** *(igual)* |

Igual que antes: **solo el masculino cambia** (mein → meinen), como *ein → einen*, *kein → keinen*.

- masc: *Das ist **mein** Pullover.* → *Ich mag **meinen** Pullover.*
- fem: *Das ist **meine** Jacke.* → *Ich nehme **meine** Jacke.* *(meine = igual)*
- neutro: *Das ist **mein** Hemd.* → *Ich suche **mein** Hemd.*
- plural: *Das sind **meine** Schuhe.* → *Ich mag **meine** Schuhe.*

---

## El punto que confunde: `sein` (de él) vs `ihr` (de ella)

El posesivo depende del **DUEÑO**, no del objeto poseído:

- **Peter** (er) → **sein**: *Peter sucht **seinen** Schuh.* (su zapato — de Peter)
- **Anna** (sie) → **ihr**: *Anna sucht **ihre** Jacke.* (su chaqueta — de Anna)

Ojo: *„ihr"* también significa "de ellos" y, con mayúscula, *„Ihr"* = "de usted".

---

## Errores a evitar
- No olvides la **-en** en masculino Akkusativo: *„Ich mag ~~mein~~ Pullover"* → **meinen** Pullover.
- Femenino con **-e** (¡la misma de *eine/keine*!): *„~~mein~~ Jacke"* → **meine** Jacke.
- *sein/ihr* según el dueño, no según el objeto.
