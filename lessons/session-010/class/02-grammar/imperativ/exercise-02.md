# Ejercicio 02 — imperativo *ihr* y *Sie* + separables (12 ítems)

## A) Forma *ihr*
> **Bilde den Imperativ mit „ihr".** *(Forma el imperativo de "vosotros/ustedes".)*

1. kommen → __________ !
2. warten → __________ !
3. sprechen → __________ !  *(¿cambia la vocal en *ihr*?)*
4. sein → __________ !

## B) Forma *Sie* (formal)
> **Bilde den höflichen Imperativ mit „Sie".** *(Forma el imperativo cortés con "Sie".)*

5. kommen → __________ !
6. nehmen → __________ !
7. mir helfen → __________ __________ ! *(„¿me ayuda usted?" en orden cortés → „Ayúdeme")*
8. sein → __________ ! *(„____ ____ so nett!" = ¡Sea tan amable!)*

## C) Verbos separables (¡prefijo al final!)
> **Bilde den Imperativ mit „du".** *(Forma el imperativo de "tú"; separa el prefijo.)*

9. einkaufen (ein|kaufen) → __________ bitte __________ !
10. aufstehen (auf|stehen) → __________ schnell __________ !
11. zurückgeben (zurück|geben) → __________ die Jacke __________ !  *(geben: e→i)*
12. anprobieren (an|probieren) → __________ das Kleid __________ !
