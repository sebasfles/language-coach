# Tarea 01 (escritura) — diálogo en el supermercado

> **Registro: diálogo.** **Schreib einen kurzen Dialog** *(escribe un diálogo corto)* de **6–8 líneas** entre **tú (Kunde/Kundin)** y un **Verkäufer / eine Verkäuferin** en el supermercado.

## La situación
Buscas una **chaqueta (die Jacke)** y no encuentras la talla. Pides ayuda, preguntas el precio, y al final pagas.

## Debes incluir (al menos)
- Pedir ayuda con un **modal + du/Sie**: *Können Sie mir helfen?* / *Kannst du mir helfen?*
- Un **imperativo** (¡el tema de hoy!): p. ej. *Zeigen Sie mir…* / *Geben Sie mir…* / *Kommen Sie…*
- Preguntar el **precio**: *Was kostet…?* / *Wie viel kostet…?*
- Un **acusativo masculino** correcto (*einen Korb*, *den Kassenbon*).
- Cómo **pagas**: *bar* o *mit Karte*.
- Al menos una **conjunción** (*und / aber / oder / denn*).

## Consejos
- Puedes usar vocab de las sesiones 8–10 (Größe, Kassenbon, reduziert, Bargeld, Wechselgeld…).
- **Atrévete a experimentar** — si quieres decir algo que no sabes, inténtalo; fallar aquí vale oro.
- Marca quién habla: *Kunde:* / *Verkäufer:*.
