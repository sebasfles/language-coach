# 02-grammar — `dieser` / `welcher` (los der-words) · sesión 11

> Roadmap A1 #14. Referencia cumulativa: `grammar/dieser-welcher.md`.
> **La idea clave:** hay un grupo de palabras que **declinan EXACTAMENTE como el artículo `der/die/das`**. Si sabes el artículo, sabes la terminación. Por eso se llaman **der-words**.

---

## 1. Qué son los der-words

Palabras que acompañan al sustantivo y toman **las mismas terminaciones que `der/die/das`**:
- **`dieser`** = este/esta/esto
- **`welcher`** = ¿cuál? / ¿qué? (para **preguntas**)
- **`jeder`** = cada / todo (solo singular)
- (+ *mancher* "algunos", *solcher* "tal" — para reconocer)

## 2. La tabla (Nominativo y Akkusativo)

Compara: la terminación del der-word **copia** la del artículo definido.

| caso | masc | fem | neut | plural |
|---|---|---|---|---|
| **Nom** | de**r** / diese**r** / welche**r** | di**e** / dies**e** / welch**e** | da**s** / diese**s** / welche**s** | di**e** / dies**e** / welch**e** |
| **Akk** | de**n** / diese**n** / welche**n** | di**e** / dies**e** / welch**e** | da**s** / diese**s** / welche**s** | di**e** / dies**e** / welch**e** |

> **El único cambio Nom→Akk es el masculino** (igual que ya sabes: *der→den*): **dieser→diesen, welcher→welchen**. Fem/neut/plural no cambian. 🎯 (Esto es tu weak-spot *der→den* otra vez, con otra ropa.)

## 3. `welcher` — preguntar "¿cuál?" (clave para transporte)

Concuerda con el género del sustantivo por el que preguntas:
- **Welcher** Bus fährt zum Zentrum? *(der Bus → welcher, Nom)*
- **Welche** Linie nehme ich? *(die Linie → welche)*
- **Welches** Gleis? *(das Gleis → welches)*
- **Welchen** Zug nimmst du? *(der Zug → welchen, Akkusativo: es objeto)*

## 4. `dieser` — "este"

- **Dieser** Zug fährt nach Berlin. *(masc Nom)*
- Ich nehme **diesen** Bus. *(masc Akk → diesen)*
- **Diese** U-Bahn ist voll. *(fem)* · Ich kaufe **dieses** Ticket. *(neut Akk = neut Nom)*

## 5. `jeder` — "cada" (singular)
- **Jeder** Zug hat heute Verspätung. · **Jede** Linie fährt zum Bahnhof.

---

## Resumen de bolsillo
- der-words = **misma terminación que der/die/das**.
- Nom→Akk cambia **solo el masculino**: *dieser→diesen*, *welcher→welchen* (como *der→den*).
- **welcher** = ¿cuál?, concuerda en género: *welcher/welche/welches* (+ *welchen* si es masc. objeto).
