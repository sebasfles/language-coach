## La idea grande

En alemán, el **artículo cambia según la función** de la palabra en la frase (eso es "el caso"). Ya usas el **Nominativo** (el sujeto: quién hace la acción). Hoy entra el **Akkusativ**: el **objeto directo** — la cosa/persona que **recibe** la acción.

- **Nominativo (sujeto):** *Der Salat ist frisch.* (¿quién/qué es? → el salat)
- **Akkusativo (objeto directo):** *Ich esse **den** Salat.* (¿qué como? → el salat, que recibe la acción)

### 🎁 La gran noticia: SOLO el masculino cambia

Este es el regalo del Akkusativ para un principiante:

| | Nominativo | **Akkusativo** |
|---|---|---|
| **masculino** | der / ein / kein | **den / einen / keinen** ← ¡cambia! |
| femenino | die / eine / keine | die / eine / keine *(igual)* |
| neutro | das / ein / kein | das / ein / kein *(igual)* |
| plural | die / — / keine | die / — / keine *(igual)* |

> Regla en una línea: **`der → den`**, **`ein → einen`**, **`kein → keinen`**. Todo lo demás **no cambia**. (Pista de memoria: la **-n** "masculina" del Akkusativ.)

## En contexto — pedir en el restaurante (tema de hoy)

Casi todos los verbos de comer/pedir llevan **objeto directo → Akkusativ**: *nehmen, essen, trinken, möchten, bestellen, haben, empfehlen*.

- **masc:** *Ich nehme **den** Salat.* · *Ich möchte **einen** Kaffee.* · *Ich trinke **einen** Saft.*
- **fem:** *Ich esse **eine** Suppe.* · *Ich möchte **die** Rechnung.* *(die = igual que en Nominativo)*
- **neutro:** *Ich trinke **ein** Bier.* · *Ich nehme **das** Brot.* *(ein/das = igual)*
- **plural:** *Ich möchte **die** Brötchen.* *(die = igual)*

👉 Fíjate: solo cuando la palabra es **masculina** tienes que "activar" el cambio (den/einen). Con die/das/plural, **escribes igual que en Nominativo**.

## Los pronombres en Akkusativ (versión ligera)

Los pronombres personales también tienen forma de Akkusativ:

| Nominativo | Akkusativo |
|---|---|
| ich | **mich** |
| du | **dich** |
| er | **ihn** |
| sie (ella) | sie |
| es | es |
| wir | **uns** |
| ihr | **euch** |
| sie / Sie | sie / Sie |

- *Der Kellner sieht **mich**.* (el camarero me ve a mí)
- *Ich sehe **dich**.* (te veo)
- *Kennst du **ihn**?* (¿lo conoces? — *ihn* = él como objeto)

## Errores típicos a evitar (español/inglés)

- No olvides el cambio masculino: *„Ich nehme ~~der~~ Salat"* → **den** Salat.
- Con femenino/neutro **no inventes** un cambio: *„Ich esse ~~eina~~ Suppe"* → **eine** Suppe (igual que en Nominativo).
- El género manda: si no sabes si es der/die/das, no sabrás si cambia — por eso seguimos afianzando el género (tu weak-spot).
