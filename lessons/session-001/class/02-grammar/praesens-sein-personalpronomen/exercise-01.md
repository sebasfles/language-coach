## Parte A — Completa con `sein` (bin / bist / ist / sind / seid / sind)

Rellena el hueco con la forma correcta del verbo `sein`.

2. Du ____ aus Spanien.
3. Maria ____ nett.
4. Wir ____ in Berlin.

6. Sie (ustedes) ____ willkommen.

## Parte C — Pon las palabras en orden (regla V2: el verbo va en posición 2)

Ordena los elementos para formar una oración correcta. Empieza la frase por la palabra que aparece **subrayada**.

13. komme / ich / aus Spanien  → (empieza por **ich**)
14. ich / heute / müde / bin  → (empieza por **heute**)  ·  *pista: con "heute" delante, ¿dónde va el sujeto?*

## Parte E — Elige la opción correcta (distinguir formas que se confunden)

Marca **una** opción. Cada distractor es un error real típico.

19. ¿"De dónde eres?" en alemán informal es…
   - a) Wo kommst du?
   - b) Woher kommst du?
   - c) Woher du kommst?
