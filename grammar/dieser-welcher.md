# dieser / welcher — los der-words (referencia cumulativa)

> Página de referencia que crece con las sesiones. Introducido en **sesión 11** (roadmap A1 #14).
> Estado en `grammar.md`. Explicación de la sesión: `lessons/session-011/class/02-grammar/dieser-welcher/grammar.md`.

## Qué son
Determinantes que declinan **como el artículo definido `der/die/das`**: **dieser** (este), **welcher** (¿cuál?), **jeder** (cada), *mancher, solcher*.

## Tabla Nom / Akk
| caso | masc | fem | neut | plural |
|---|---|---|---|---|
| Nom | dieser / welcher / jeder | diese / welche / jede | dieses / welches / jedes | diese / welche (jeder no tiene plural) |
| Akk | diesen / welchen / jeden | diese / welche / jede | dieses / welches / jedes | diese / welche |

**Regla de oro:** copia la terminación de `der/die/das`. **Solo el masculino cambia Nom→Akk** (dieser→diesen, welcher→welchen, der→den).

## `welcher` (preguntas "¿cuál?")
Concuerda con el género del sustantivo: *Welcher Bus? · Welche Linie? · Welches Gleis? · Welchen Zug nimmst du?* (masc objeto → welchen).

## `dieser` (este)
*Dieser Zug · diesen Bus (Akk) · diese U-Bahn · dieses Ticket.*

## Trampas para hispanohablante
- Es el **mismo** patrón que *der→den*: si dominas el artículo, dominas el der-word. El error típico es olvidar la *-n* del masculino acusativo (*diesen/welchen*).
- *welches* (neutro) se confunde con *welche* (fem/pl) — mira el género del sustantivo.
