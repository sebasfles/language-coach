# Referencia — Akkusativ (el caso del objeto directo)

> Página de referencia **acumulativa**. Roadmap A1 #8.
> Estado en `grammar.md`. Contenido en español; el alemán es el idioma meta.

## Regla base

El **Akkusativ** marca el **objeto directo** (lo que recibe la acción). **Solo el masculino cambia el artículo:**

| | Nominativo | **Akkusativo** |
|---|---|---|
| masculino | der / ein / kein | **den / einen / keinen** |
| femenino | die / eine / keine | die / eine / keine *(igual)* |
| neutro | das / ein / kein | das / ein / kein *(igual)* |
| plural | die / — / keine | die / — / keine *(igual)* |

Regla mnemónica: **der→den, ein→einen, kein→keinen** (la "-n" masculina). El resto no cambia.

## Pronombres personales en Akkusativo

| Nom | Akk |
|---|---|
| ich → **mich** | wir → **uns** |
| du → **dich** | ihr → **euch** |
| er → **ihn** | sie/Sie → sie/Sie |
| sie → sie · es → es | |

Ej.: *Ich sehe **dich**. · Kennst du **ihn**? · Der Kellner sieht **uns**.*

## Verbos frecuentes con objeto directo (Akkusativ)

*nehmen, essen, trinken, möchten, bestellen, haben, sehen, empfehlen, kaufen, brauchen* → *Ich nehme **den** Salat / **einen** Kaffee / **eine** Suppe / **ein** Bier.*

## Historial

- **Sesión 5 (introduced@5):** Akkusativ de los artículos (definido, indefinido, kein) con la regla "solo el masculino cambia" (der→den, ein→einen), en el contexto de pedir en el restaurante; + introducción de los pronombres en Akkusativ. Motivado por la escritura de la s4 (faltaba *„den Fisch"*).
